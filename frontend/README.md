# Halleyx Dashboard Builder

A custom dashboard builder built with React + Vite.

## Quick Start

```bash
npm install
npm run dev
```

## Stack
- React 18
- Vite 5
- Chart.js + react-chartjs-2
- CSS Modules

## Features
- Drag & drop widget placement onto a 12-column canvas
- Free drag to rearrange, corner handle to resize
- Widget types: Bar, Line, Area, Pie, Scatter charts · Table · KPI card
- Per-widget config panel (axes, colors, columns, pagination, filters)
- Customer Orders CRUD (create, edit, delete, right-click context menu)
- Date range filter applied to all dashboard data
- Responsive: 12 → 8 → 4 column breakpoints
